# 体高

## 协议须知

|规则|描述|
|:-|:-|
|接口地址|height|
|传输方式|HTTP|
|请求方法|GET或POST|
|字符编码|统一采用UTF-8编码|
|响应格式|统一采用JSON格式|
|接口鉴权|签名机制，详情请阅接口鉴权|
